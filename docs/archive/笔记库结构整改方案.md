## 三、注意事项

1. **两个 commit 分开提交** — mv 一个，配置更新一个，git log 清晰，回滚方便
2. **Obsidian 的 `alwaysUpdateLinks: true`** 已开启，wikilink 会自动更新，但 Dataview `FROM` 路径和 CLAUDE.md 中的硬编码路径**不会自动更新**
3. **执行前先 `git stash` 或确认工作区干净**，避免 mv 和未提交改动混在一起
4. **SAP 模块知识/FICO/由浅入深学习SAP财务/** 作为 FICO 子目录会随 FICO 一起搬走，无需单独处理
5. **执行后在 Obsidian 中打开主页和文档索引**，检查链接是否正常，Obsidian 会标红断链
