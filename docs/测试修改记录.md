### 设计决策解释

**为什么任务/想法/项目/灵感/写作/学习都保存到 `选题/` 文件夹？**

这是设计文档的约定：`选题/` 是统一的待办事项池，通过 frontmatter 的 `type` 字段区分类型（任务/想法/项目/灵感/写作/学习），通过 `status` 字段跟踪状态（待评估/进行中/已完成/已放弃）。这样做的好处：
- 统一管理，不需要维护多个文件夹
- Dataview 查询方便，一个文件夹就能筛选所有类型
- 文件名格式 `YYYY-MM-DD-标题.md` 自带创建日期

function toLocalDateStr(date: Date): string {
  const y = date.getFullYear();
  const m = String(date.getMonth() + 1).padStart(2, "0");
  const d = String(date.getDate()).padStart(2, "0");
  return `${y}-${m}-${d}`;
}
